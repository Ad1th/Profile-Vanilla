# Personal Portfolio Website – Adith Manikonda

## Overview
This is a modern, responsive personal portfolio website for Adith Manikonda, an engineering student at VIT. The site showcases skills, projects, experience, achievements, and more, serving as a digital resume and interactive showcase for recruiters and collaborators.

## Features
- **Section-based navigation**: About, Experience, Skills, Projects, Patents, Achievements, Hackathons, Hobbies, Contact
- **Responsive design**: Mobile-friendly layout with a dedicated mobile menu
- **Modern UI**: Glassmorphism, custom fonts, and smooth animations
- **Dynamic content**: JavaScript-driven population of experience, skills, projects, etc.
- **Accessibility**: Semantic HTML, ARIA labels, and keyboard navigation
- **Performance**: Optimized font loading, minimal dependencies, fast load times

## Technology Stack
- **HTML5**: Semantic structure for all sections
- **CSS3**: Custom styles, Flexbox, Grid, glassmorphism, and animations
- **JavaScript (ES6+)**: Dynamic content, menu toggling, scroll progress, and reveal animations

## Folder Structure
```
Profile-Vanilla/
├── index.html
├── style.css
├── script.js
├── public/
└── README.md
```

## How to Run
1. Clone the repository or download the source code.
2. Open `index.html` in your browser.
3. All assets are local; no build step is required.

## Customization
- Update content in `index.html` and JavaScript data arrays in `script.js` for Experience, Skills, Projects, etc.
- Modify styles in `style.css` for branding or color changes.

## Accessibility & SEO
- Uses semantic HTML and ARIA labels for accessibility.
- Meta tags for description and viewport for SEO.
